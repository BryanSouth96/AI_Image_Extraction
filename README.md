# AI_Image_Extraction

Extract information from images passed to GPT models via Azure OpenAI API with this code.

**Note:** While I have not used public Github during my five years at Statistics Canada, I do use the internal Github at Statistics Canada. Because the data we work with is sensitive I cannot share full project repos, but I can share this example code of how I pass images to AI models for extraction. I have also provided an example prompt taken from a recent project which was iterated on until the required accuracy was achieved.
